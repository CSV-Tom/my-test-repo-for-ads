# Git Commands

## List of commands

```
git init
git status
git log
git add README.md
git commit -m "<message>" 
git checkout .
git checkout -b feature_x
git diff
git diff HEAD~1

git checkout -b feature_a
git commit -m "<message>" README.md
git checkout -b feature_b
git commit -m "<message>" README.md

git blame README.md
git diff HEAD~2
git diff master..feature_a

git branch -av
git push -u origin master
git push -u origin feature_tom
git pull
git fetch
```
